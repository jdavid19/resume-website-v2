Project Overview


A personal resume website created as a static website and hosted on AWS with continuous deployment. The project demonstrates skills in AWS services and cloud infrastructure.


AWS Services Used:

  • AWS CodePipeline: For CI/CD, pulling updates directly from GitHub to automatically deploy changes.
  • Amazon S3: Static website hosting for storing and serving the website content.
  • Amazon Route 53: DNS management to handle domain routing.
  • AWS Certificate Manager (ACM): SSL certificate provisioning for secure HTTPS.
  • AWS API Gateway, Lambda, and DynamoDB: Integrated page view counter to track visitor counts.
  • Amazon CloudFront: Content Delivery Network (CDN) for faster content distribution.
